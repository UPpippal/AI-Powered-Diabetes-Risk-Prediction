# 🩺 AI-Powered Diabetes Risk Prediction System

An end-to-end Machine Learning healthcare project for predicting diabetes risk using patient health indicators and Generative AI-based health recommendations.

## 🚀 Features

- Diabetes Risk Prediction using Machine Learning
- Streamlit-based Interactive Web App
- SMOTE for Class Imbalance Handling
- Hyperparameter Tuning
- AI Health Assistant for Personalized Guidance
- Probability-based Risk Prediction

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Google Gemini API
- Joblib

## 📊 Machine Learning Workflow

1. Data Cleaning & Preprocessing  
2. Exploratory Data Analysis (EDA)  
3. Feature Engineering  
4. Class Imbalance Handling using SMOTE  
5. Model Training & Evaluation  
6. Hyperparameter Tuning  
7. Streamlit Deployment  
8. AI Health Assistant Integration

## 📈 Model Performance

- Model: Logistic Regression
- Recall Score: ~76%
- Healthcare-focused optimization
